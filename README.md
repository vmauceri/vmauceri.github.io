# Spend Less, Sea More

This is a simple static website with a home page and 5 subpages, where 4 of the subpages have one subpage each.

## Structure

- Home (index.html)
- Travel North America/Canada (page1.html)
  - Cruising through Hawaii's Islands on a Real-Life Budget (page1/subpage1_1.html) - has a direct link to home
- Travel Asia (page2.html)
  - Travelling Thailand on a Real-Life Budget (page2/subpage2_1.html)
- Travel Europe (page3.html)
  - Travelling Georgia on a Real-Life Budget (page3/subpage3_1.html)
- Travel the Middle East (page4.html)
  - Travelling Saudi Arabia on a Real-Life Budget (page4/subpage4_1.html)
- Closing page (page5.html) - no subpages

## Usage

Open index.html in a web browser to view the site.

## Deployment

This site is set up for GitHub Pages. Push to the main branch to deploy.
